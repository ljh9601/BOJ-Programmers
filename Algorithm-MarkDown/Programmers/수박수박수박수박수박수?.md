

## Lv1. 연습문제 - 수박수박수박수박수박수?

<br>



![](https://i.imgur.com/BkuM5XM.png)

<br>

> 매우 간단한 문제다. 짝수번째 인덱스는 '수', 홀수번째 인덱스는 '박' 을 더해주면 되는데, 필자의 경우 '수박'을 n번 더한 후 n번째 인덱스 전까지만 반환하도록 했다! 이유는 간단하다.
>
> 그저 숏코딩을 위함이다..ㅋㅋ

<br>

소스코드는 다음과 같다.

```python
def solution(n):
    return ('수박' * n)[:n]
```
